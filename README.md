### 프로젝트 피드백
- 오라클 클라우드 사용 시 서버의 과부하가 생겨 프로젝트 구현 단계에서 테스트 진행이 매끄럽지 못했다. DB의 Connection을 닫아 자원을 해제해주는 것의 중요성을 깨달았다.
- 팀원간의 원활한 의사소통을 위한 공통 템플릿 제작의 중요성을 깨달았다.
- Github과 Notion을 통해 효율적으로 협업하는 방법을 알게 되었다.
